# SauceDemoProject
## :rocket:Project Description
This project automates the testing of the SauceDemo web application using Selenium WebDriver and TestNG.
It follows the Page Object Model (POM) design pattern to maintain clean and reusable test code. 
## :pushpin:Technologies Used
- Java
- Selenium WebDriver
- TestNG
- Maven
- POM(Page Object Model)
## :file_folder:Project Structure
src/main/java
    base->baseTest
    pages → Page classes (LoginPage, ProductPage, CartPage, CheckoutPage, CheckoutInfoPage, LogoutPage)
src/test/java
    tests → Test classes
## :test_tube:Test Cases Implemented
- Login Test
- Add to Cart Test
- Checkout Flow Test
- Product Sorting Test
 ## :arrow_forward:How to Run the Tests
1.Clone the repository
2.Open the project in Eclipse 
3.Run the TestNG XML file
## :gear:Framework Features
- Page Object Model design
- Reusable methods
- TestNG annotations
- Separate test classes
